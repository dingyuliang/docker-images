# Docker Image for .NET Framework 4.8-3.5

## Features
1. .NET Framework 4.8-3.5
2. .NET Framework v3.5 Client Profile (to support ClickOnce project)
3. All Nuget Packages from packages.config
4. nunit3-console.exe
5. OpenCover.Console.exe
6. 7za.exe
7. Common CICD Powershell script library (see ./tools/.ps/Modules/CICD.psm1)
